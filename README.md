# TIẾNG TRUNG MIA

Ứng dụng học và ôn luyện từ vựng HSK, flashcard, trắc nghiệm, luyện gõ chữ Hán và thứ tự nét viết.

## Website

https://tunguyen29121997-design.github.io/tieng-trung-mia/

Các tệp production được đặt ở thư mục gốc để GitHub Pages phục vụ trực tiếp từ nhánh `main`.

## Mã nguồn

Mã React/Vite nằm trong thư mục `source-code/`.

```bash
cd source-code
npm install
npm run dev
```

Tạo bản production:

```bash
npm run build
```
